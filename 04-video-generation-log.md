# 04. 비디오 생성/변환 로그

## 사용 도구 계획

| 도구 | 목적 |
|---|---|
| Pika 또는 Runway | 이미지 기반 짧은 모션 컷 생성 |
| Kling 또는 Canva | 대체 모션 제작 도구 |
| CapCut 또는 Canva | 컷 편집, 자막, 오디오 통합 |
