# 03. Image Generation Log

This document records the actual image generation process for the VERI AUTONOMY advertisement project.

## 1. Purpose

The project creates six key visual scenes for a 42-second brand advertisement video.
These images will later be transformed into short motion clips using an image-to-video or video generation tool.

## 2. Project Context

| Item | Description |
|---|---|
| Brand | VERI AUTONOMY |
| Campaign | Verified Intelligence for Safer Autonomy |
| Ad Goal | Brand awareness and website visit |
| Main Message | Connect complex AI, autonomy, and technical knowledge into a more trustworthy and structured form |
| Character Usage | Prupy is used as a private visual reference for mood and identity |
| Repository Rule | Original logo, original character source, and original 3D source files are not uploaded to this repository |

## 3. Visual Direction

| Item | Description |

## 5. Prompt Strategy

The image generation strategy was to avoid unreliable AI-generated text.

For scenes requiring brand name, slogan, CTA, or value labels, the image was generated as a clean background first.
Accurate text will be added later in Canva or CapCut.

Common style direction:

Minimal premium technology brand advertisement, black white and cyan blue palette, clean futuristic interface, research-driven mood, engineering precision, trustworthy atmosphere, soft cinematic lighting, 16:9 aspect ratio.
